Auf dem Tisch befinden sich 21 Spielkarten verdeckt in einer Fensteranordnung. Die 4 Eckkarten sind bereits aufgedeckt. Die aufgedeckten Karten deklarieren den Start.

Sie können an einer beliebigen aufgedeckten Karte anfangen, eine benachbarte Karte aufzudecken, wobei sie bestimmen müssen, ob die Karte im Wert höher, niedriger oder gleich ist im Bezug auf die bereits aufgedeckte Karte.

Stimmt ihre Aussage, dürfen sie dies beliebig Oft an jeder beliebigen aufgedeckten Karte wiederholen. Sie können ihren Zug nach mindestens einem erfolgreichem Aufdecken beenden, dann ist der nächste Spieler am Zug.

Stimmt eine Aussage nicht, werden alle aufgedeckten Karten die direkt benachbart sind, inklusive der, an der ihr Zug gescheitert ist, ausgetauscht und wieder verdeckt, und ihr Zug beginnt von neu.

Für jede in ihrem Zug ausgetauschte Karte bekommen sie eine Strafe.

Liegt eine verdeckte Karte zwischen zwei oder drei offenen Karten wird zwischen innerhalb oder außerhalb im Bezug auf den kleinsten und größten der Werte entschieden, beziehungsweise gleich zu einem der beliebigen Werte.

Sollte das Fenster von einem Spieler komplett geöffnet werden, darf dieser 42 Strafen unter seinen Mitspielern aufteilen.
